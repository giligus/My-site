# Personal Portfolio Website

A clean static website for presenting a personal introduction, experience, portfolio projects, and contact links.

## Sections

- Hero introduction
- Professional highlights
- About me
- Experience timeline
- Portfolio cards
- Contact links

## Customize

- Update the name, intro, experience entries, portfolio cards, and contact links in `index.html`.
- Replace `assets/hero-workspace.png` with your own image if you want a more personal visual.
- Point the portfolio links to the GitHub repositories once they exist.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Deploy on GitHub Pages

1. Create a GitHub repository.
2. Push this folder to the repository.
3. In GitHub, open **Settings > Pages**.
4. Set the source to the default branch and root folder.
